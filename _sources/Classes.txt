.. cpp:class:: ph_mode
    :noindex:

    | **double** qx,qy,qz; - q-point coordinates (in crystal coordinates).  
    | **double** \*omega_q\; - frequencies of the various polarizations at this q-point.  
    | **double** \**eig_vec_q_real\; - real part of the eigenvectors at this q-point.  
    | **double** \**eig_vec_q_image\; - imaginary part of the eigenvectors at this q-point.  
    | **double** \*vg_x\; - x-component of group velocity at this point.
    | **double** \*vg_y\; - y-component of group velocity at this point
    | **double** \*vg_z\; - z-component of group velocity at this point
    | **double** \*phase_space_abs\; - phase space for phonon absorption processes
    | **double** \*phase_space_emi\; - phase space for phonon emission processes
    | vector **<int>** \*mode_index1\; - vector of modes that interact with this mode 
    | vector **<int>** \*pol_index1\; -  vector of the corresponding polarizations
    | vector **<int>** \*mode_index2\; - vector of modes that interact with this mode
    | vector **<int>** \*pol_index2\; -  vector of the corresponding polarizations
    | vector **<char>** \*scat_type\; - vector that stores the scattering type (absorption or emission)
    | vector **<double>** \*matrix_element_sq\; - vector that stores the square of matrix element
    | vector **<double>** \*broadening\; - vector that stores the broadening
    | **double** \*f_old\; - phonon occupation function 
    | **double** \*f_new\; - new occupation function
    | vector **<int>** \*mode_kindex\; - kpt of interaction
    | vector **<int>** \*mode_kqindex\; - k+qpt of interaction
    | vector **<int>** \*e_band_index\; - band of the kpt of the interaction
    | vector **<int>** \*e_band_index_kq\; - band of the k+qpt of the interaction
    | vector **<double>** \*eph_matrix_element\; - matrix element
    | vector **<int>** \eph_mat_array_index\; - vector containing the indices where this phonon mode occurs in the eph array
